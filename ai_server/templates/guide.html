Terrain Generator Guide

Follow this guide to create the perfect terrain by describing what features to include, what to avoid, and how to set up the environment.

1. Basic Terrain Input

The following parameters must be provided for any terrain:

  • Width & Height: Specify the size of the terrain (e.g., 512 to 1024 units).
  • Depth: Controls the maximum height. Enter a value between 65 and 200 units.
  • Octaves: The level of detail. Use values between 1 and 15.
  • Scale: Controls the size of terrain features. Use values between 70 and 500.

2. Describing Your Terrain

Use descriptive terms to specify the type of terrain you want to generate. Some examples:

  • Desert: No trees or grass unless specifically requested.
  • Mountain: Adds elevation with rocky textures and optional snow.
  • Forest: Includes grass and trees of varying density.
  • Snow: Only snow textures unless additional textures are requested.

If you want certain objects removed (e.g., "no trees, no grass"), specify this in your description, and the terrain generator will adjust accordingly.

3. Textures and Surface Features

  • Allowed Textures: grass, desert, snow, mud, rock, sand, forest floor, mountain rock, dirt, dead grass.
  • Multiple Textures: Mention multiple textures if you want a combination (e.g., "desert with patches of dead grass").

Each texture will be adjusted based on height curves, with smooth transitions between textures.

4. Adding Grass or Trees

When adding vegetation, the following rules apply:

  • Grass Density: Higher values (700 to 1000) mean more dense grass.
  • Tree Density: Use low density (max 0.3) for sparse forests or high density (1) for dense forests.
  • Steepness Limits: Grass or trees will not appear on slopes steeper than 90 degrees.
  • Height Range: Vegetation will grow only within specified height limits (e.g., -200 to 200 units).

5. Special Cases: Cities

If you mention a city or city-like features, use the following options:

  • City Size: Choose from Small, Medium, Large, or Very Large.
  • Satellite City: Specify if the city should include satellite areas connected by highways.
  • Downtown Area: Indicate if the city should have a downtown metropolitan area.
  • Traffic System: Add a traffic system if the city needs to simulate vehicles and movement.
  • Traffic Hand: Specify "RightHand" or "LeftHand" traffic rules.

6. Water Features

Water can be added to the terrain in different forms:

  • Water Types: Choose from river, lake, or ocean. If you don't want water, specify "none".
  • Water Level: For lakes or oceans, specify the water height (e.g., between 50 and 0).
  • River Width: Specify the width range (e.g., 100 to 1000 units).

7. Atmosphere and Time of Day

Customize the environment by specifying the time of day and weather effects:

  • Time of Day: Use values from 0 (midnight) to 24 (midnight). For example, 12 is noon.
  • Sun Size: Enter a value between 0 and 1 (default is 0.05).
  • Fog Intensity: Use values between 0 (no fog) and 0.5 (dense fog).
  • Sky Tint: Provide RGB values for the sky color (e.g., r=0.5, g=0.5, b=0.5).

8. Final Tips for Generating Terrains

Keep the following tips in mind when creating your terrain:

  • Randomization: Enable randomization for varied results each time the terrain is generated.
  • Use Falloff Maps: Controls smoothness at terrain edges, usually true.
  • Multiple Objects: Add rocks, trees, or other elements by describing their distribution.
  • Precision Descriptions: The more specific your description, the more accurate the generated terrain will be.
